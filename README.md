# triple-backend

Triple tourist club mileage service

[Table/API 명세서](https://github.com/gar-den/triple-backend/blob/main/description.md)

실행 방법
```
> yarn config set ignore-engines true
> yarn install
> yarn db:create
> yarn db:gen
> yarn server
```

테스트 코드 실행 방법
```
> yarn install
> yarn db:create
> yarn test
```
