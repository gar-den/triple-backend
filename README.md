# triple-backend

Triple tourist club mileage service

[Table/API 명세서-notion ver](https://wirehaired-snapper-f57.notion.site/API-cf415bdf56ab4c20926e3bfcb9ccff9f)

실행 방법
```
> yarn config set ignore-engines true
> yarn install
> yarn db:create
> yarn db:gen
> yarn server
```

테스트 코드 실행 방법
```
> yarn install
> yarn db:create
> yarn test
```
