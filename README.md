# triple-backend

Triple tourist club mileage service

[Table/API 명세서](https://github.com/gar-den/triple-backend/blob/main/description.md)

### Specification
- Typescript 4.7.4
- Express 4.17.13
- Sequelize 6.9.0
- MySQL
- Jest
- ESLint


### 실행 방법
```
> yarn config set ignore-engines true
> yarn install
> yarn db:create
> yarn db:gen
> yarn server
```

### 테스트 코드 실행 방법
```
> yarn install
> yarn db:create
> yarn test
```
